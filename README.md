# ITX Frontend Test - Mobile Shop

Miniaplicación SPA para la compra de dispositivos móviles.

## Instrucciones de ejecución

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Modo desarrollo (START):**
   ```bash
   npm run start
   ```

3. **Compilación para producción (BUILD):**
   ```bash
   npm run build
   ```

4. **Lanzamiento de test (TEST):**
   ```bash
   npm run test
   ```

5. **Comprobación de código (LINT):**
   ```bash
   npm run lint
   ```

## Especificaciones del proyecto
- **SPA:** Implementación de navegación por parte del cliente.
- **Caché:** Sistema de almacenamiento en cliente con expiración de 1 hora.
- **Vistas:** Listado de productos (PLP) y detalle de producto (PDP).
